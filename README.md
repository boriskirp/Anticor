# Оркестратор: Антикорозія

**Розумовий простір. Всесвіт як параметр.**

Цей проект — це інструментальне середовище для дослідників.
Це не статичний текст, а інтерактивний інструмент, де виділення тексту активує знання.

## Як працювати з інструментом:
1. Перейдіть на інтерактивну сторінку за посиланням нижче.
2. Виділяйте мишкою ключові терміни (наприклад, "корозія").
3. Отримуйте миттєві інсайти в попап-вікнах.
